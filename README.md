# 浜名湖うなぎ紹介サイト

観光客向けに浜名湖うなぎの魅力を伝える静的紹介サイト。

本サイトは Kiro コーディングエージェントで作成しました（LLM: GLM-5）。プランモードでサイト構成・デザインの詳細を決定し、実装まで一気通貫で行いました。

## 概要

- **目的**: 浜名湖うなぎの歴史・特徴・お店・料理・アクセス情報を観光客に提供
- **技術**: HTML / CSS / Vanilla JS（ビルドツール不要）
- **対応**: レスポンシブデザイン（PC・スマートフォン）

## ページ構成

| ファイル | 内容 |
|---|---|
| `index.html` | トップページ（ヒーロー・サイト概要・ナビカード） |
| `history.html` | 歴史・特徴（養殖の歴史・美味しさの理由） |
| `shops.html` | お店紹介（おすすめ店舗4軒） |
| `dishes.html` | 料理紹介（蒲焼・白焼き・うな丼・ひつまぶし） |
| `access.html` | アクセス・観光情報（交通手段・地図・周辺スポット） |
| `style.css` | 共通スタイル |
| `main.js` | ハンバーガーメニュー・アクティブリンク |

## ローカルで確認

```bash
python3 -m http.server 8080
# → http://localhost:8080 をブラウザで開く
```
